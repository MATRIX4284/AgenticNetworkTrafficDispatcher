# TelecomNext Technical Routing SOP

This document defines the standard operating procedures for routing customer complaints to the correct technical department and assigning priority.

---

## Network Operations (NOC)

Route a complaint to **Network Operations (NOC)** if it mentions any of the following issues:

- Outages  
- Data Caps  
- Slow Speeds  
- Signal Drops  

These issues indicate problems with network performance, availability, or capacity.

---

## Billing & Accounts

Route a complaint to **Billing & Accounts** if it mentions any of the following issues:

- Billing  
- Overcharges  
- Contract Termination  
- Refunds  

These issues relate to customer accounts, invoicing, or payment disputes.

---

## Hardware Support

Route a complaint to **Hardware Support** if it mentions any of the following issues:

- Modems  
- Routers  
- Wiring  
- Installation  

These issues indicate problems with physical equipment or on-site setup.

---

## Priority Rule

If a complaint originates from either of the following states:

- Georgia  
- Florida  

The ticket **must be marked as `URGENT`**, as these states are classified as high-churn zones.

Priority rules apply independently of department routing.
